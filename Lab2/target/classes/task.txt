Система предназначена для хранения информации, с возможностью предоставления быстрого к ней доступа через связанные текстовые метки.
Система имеет следующий функционал:

 ++ Внесение новой записи в систему (принимает наименование записи(заголовок), текст записи, возвращает идентификатор записи)

 ++ Добавление меток к записи (принимает идентификатор записи, массив идентификаторов меток, которые будут к данной записи привязаны)

 ++ Добавление файла к записи (принимает идентификатор записи, имя файла, массив байт – данный «файл» будет привязан к записи)

 ++ Добавление новой метки в систему (принимает текст новой метки, возвращает ее идентификатор)

 ++ Получение файла, связанного с записью (принимает идентификатор записи, имя файла – возвращает массив байт файла)

 ++ Удаление метки из системы (принимает идентификатор метки)

 ++ Получение списка всех меток, определенных в системе (возвращает массив записей следующей структуры – текст метки, идентификатор метки)

 ++ Получение списка всех записей системы (возвращает список записей, имеющих следующую структуру – идентификатор записи системы,
     наименование записи)

 ++ Поиск записей системы по метке (принимает идентификатор метки возвращает массив записей системы,
     которые имеют данную метку. Элементы возвращаемого массива имеют структуру -идентификатор записи, наименование записи)
